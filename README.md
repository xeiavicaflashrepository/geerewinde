# geerewinde
GPLv3 screenwriting program that's written in C and hopes to have a GTK4 and ncurses interface.

I recommended checking out ToshioCP's tutorials on [GTK4](https://toshiocp.github.io/Gtk4-tutorial/) and [GObject](https://toshiocp.github.io/Gobject-tutorial/) if you want a tl;dr or refresher.
